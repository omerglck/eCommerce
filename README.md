# E-Commerce

- Follow these steps to run this project in your local environment.

## Installation

1. **Install Project Dependencies:**

   ```bash
   yarn install
   ```

2. **Install iOS Dependencies (MacOS Only):**
   ```
   cd ios
   yarn pod
   cd ..
   ```
3. **Set Up Environment Variables:**
   ```
   API_URL=https://api.example.com
   ```

## Running the Project

### iOS

```
yarn ios
```

### Android

```
yarn android
```
